== Description ==
A simple plugin for showing Alert, Message when deactivating plugins.

No settings/ hook require. just plug and play :)

== Installation ==
1. Upload \"plugin-deactivation-notice.zip\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Enjoy
